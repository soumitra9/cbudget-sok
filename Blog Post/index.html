Click any bar above to inspect what it's made of.
O = fixed overhead (system prompt + reserved output). S = tool/MCP schemas, re-sent every turn regardless of use. H = everything accumulated from prior turns (past user messages, past reasoning, past actions, past tool observations). R = new reasoning tokens generated this turn. Out = this turn's action call or final answer text.

Formalization: B = S + H + R + O, with Ht = Ht-1 + Rt-1 + Outt-1 + (new user/observation input) — reasoning and output tokens don't vanish after being generated, they become next turn's history.
